Flow wireframe

The signature flow, end to end.

Two tracks: the sender prepares and dispatches a document; the recipient receives, reviews, and signs. This wireframe shows how the screens connect, where the state changes, and which channel each moment happens in.

Solid frame: primary screen Amber: decision / state change Dashed indigo: signature field Dashed emerald: snapped / signed
